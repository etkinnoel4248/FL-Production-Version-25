Configuration
The configuration file `config/skins.json` allows you to define your themes:
```json
{
  "themes": [
    {
      "name": "Summer Vibes",
      "skin": "Summer Skin",
      "backpack": "Summer Backpack"
    },
    {
      "name": "Winter Wonderland",
      "skin": "Snowflake Skin",
      "backpack": "Snowflake Backpack"
    }
  ],
  "timer": 60,
  "random": true
}
```
